### Added

- Light color theme
- Automatic activation when open an Assembly file.
- *.asm* and *.s* file support
- Z80 Assembly full-supported
- Provide a color theme for the Z80 Assembly and the x86-64 Assembly
- Error detector for the Z80 Assembly
